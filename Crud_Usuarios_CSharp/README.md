# CRUD de Usuários em C#

Projeto simples de CRUD de usuários desenvolvido em C# utilizando Console Application.  
O objetivo do projeto é praticar conceitos fundamentais da linguagem e da programação orientada a objetos.

## Funcionalidades

- Criar usuário
- Listar usuários
- Buscar usuário por ID
- Editar usuário
- Deletar usuário

## Tecnologias

- C#
- .NET
- LINQ
- Collections (List<T>)
