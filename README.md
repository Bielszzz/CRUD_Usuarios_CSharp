# 🧑‍💻 CRUD de Usuários em C#

Projeto simples de **CRUD (Create, Read, Update, Delete)** de usuários desenvolvido em **C#** utilizando **Console Application**.

O objetivo deste projeto é praticar conceitos fundamentais da linguagem **C#**, incluindo **Programação Orientada a Objetos**, manipulação de listas e uso de **LINQ**.

---

## 📌 Funcionalidades

O sistema permite realizar as seguintes operações:

* ➕ Criar usuário
* 📋 Listar usuários
* 🔎 Buscar usuário por ID
* ✏️ Editar usuário
* ❌ Deletar usuário

---

## 🛠 Tecnologias utilizadas

* **C#**
* **.NET**
* **LINQ**
* **Collections (List)**

---

## ▶️ Como executar o projeto

### 1️⃣ Clonar o repositório

```
git clone https://github.com/Bielszzz/CRUD_Usuarios_CSharp.git
```

### 2️⃣ Entrar na pasta do projeto

```
cd CRUD_Usuarios_CSharp
```

### 3️⃣ Executar o projeto

```
dotnet run
```
